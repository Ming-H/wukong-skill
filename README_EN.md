> **I am the Great Sage Equal to Heaven, Sun Wukong.** One Skill, seventy-two transformations. Every transformation is a person.

**Other Languages / 其他语言:** [中文](README.md)

**wukong-skill v5** is a super-skill for [Claude Code](https://claude.ai/claude-code) that transforms into all kinds of characters to keep you company -- a fortune teller, Confucius, 教员 (The Teacher), the poet Li Bai, your secret crush... Drawing engineering design principles from the Nuwa Skill Factory, it features a multi-stage pipeline, human checkpoints, a triple-verification methodology, and a multi-agent parallel architecture to ensure output quality for every transformation.

**Core philosophy: Fun comes first.** It's not a toolbox -- it's a character switcher.

## Quick Install

### Option 1: Recommended -- Using npx skills add (Global Install)

```bash
chmod +x install.sh && npx skills add Ming-H/wukong-skill
```

### Option 2: Manual Install

```bash
cd wukong-skill
chmod +x install.sh
./install.sh
```

### Option 3: Clone from GitHub

```bash
git clone https://github.com/Ming-H/wukong-skill.git
cd wukong-skill
chmod +x install.sh && ./install.sh
```

## Example Dialogues

### Transformation 14 -- 教员 (The Teacher)

```
User: I always feel like I can't finish my work, I'm really anxious.

Wukong: Alright! Old Sun transforms into "教员 (The Teacher)" to help you analyze this!

教员 (The Teacher): Now, this problem -- we need to see where the principal contradiction lies.
      You say you can't finish your work -- is there truly that much to do, or are you
      failing to grasp the key points? You can't just grab everything at once like
      "grabbing eyebrows and beard together."

      First figure it out: which ones are the principal contradictions? Which are secondary?
      Put your main energy into the principal contradictions, and the rest can wait.

      "No investigation, no right to speak." First list out -- which tasks absolutely
      must be done?

      The future is bright, but the road is winding. Don't rush, take it one step at a time.
```

## Design Philosophy (v5)

1. **Full Character-Driven**: Every transformation is a person. Not a tool -- a character.
2. **Fun First**: The core goal is to be fun. Being useful is a side effect.
3. **72 Transformations = Unified Interface**: No need to install 17 separate skills -- one Sun Wukong handles everything.
4. **Three-Channel Smart Routing**: Direct transformation / diagnostic recommendation / incremental update.
5. **Multi-Stage Pipeline + Checkpoints**: Inspired by the Nuwa Skill Factory, with two mandatory checkpoints (info confirmation and result confirmation).
6. **Triple Verification Methodology**: Cross-domain reproduction, generative power, and exclusivity.
7. **Honesty Over Perfection**: Mark insufficient data, preserve contradictions, declare limitations.
8. **Mandatory Quality Self-Check**: 6-point quality self-check before output -- cannot be skipped.
9. **Roundtable Debate**: Multiple characters debate the same topic from different perspectives.
10. **Random & Tour**: Surprise random transformations and guided onboarding for new users.

## Disclaimers

- Divination transformations (1-7): Results are for entertainment purposes only.
- Self-discovery transformations (35-40): Test results do not constitute psychological diagnosis.
- Distillation transformations (20-28): All data is processed locally only.
- Emotional companionship transformations (29-34): Cannot replace professional psychological counseling.
- This skill does not collect, upload, or transmit any user data.

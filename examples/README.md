# examples/

这里是独立、自包含的名人视角 Skill 示例。每个示例可脱离 wukong-skill 单独使用。

## 与 `personas/` 的区别

| 目录 | 定位 | 用途 |
|------|------|------|
| `personas/` | 72变的内部角色声音文件 | 被 SKILL.md 在变身时按需读取 |
| `examples/` | 完整的独立 Skill 包 | 教用户如何创建自己的名人视角 Skill |

## 内容重叠说明

`personas/jobs.md` 和 `personas/taleb.md` 是简化版人设（身份卡 + 表达DNA + 心智模型 + 典型对话流），服务于第51变/第52变的快速角色切换。

`examples/` 中的 Skill 包含完整的研究素材（`references/research/` 下的多 Agent 调研文件）和结构化的 SKILL.md，是由第二十八变「女娲造物」生成的成品示例。

两者共享核心人设信息，但定位不同：`personas/` 是 wukong 内部使用的声音，`examples/` 是可独立分发的完整 Skill。

## 当前示例

- `steve-jobs-perspective/` — Steve Jobs 视角（第51变对应的完整版）
- `taleb-perspective/` — Nassim Taleb 视角（第52变对应的完整版）

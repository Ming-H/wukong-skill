# 第三变·遁甲 — 奇门遁甲

> 以时间起局，排天盘地盘八门九星八神，通过用神判断预测事物发展趋势。

## 起局方法

### 确定遁甲类型

| 类型 | 时间范围 | 局数 |
|------|----------|------|
| 阳遁一局 | 冬至后第1候 | 阳遁1局 |
| 阳遁九局 | 冬至后各候依次 | 阳遁1-9局 |
| 阴遁九局 | 夏至后第1候 | 阴遁9局 |
| 阴遁一局 | 夏至后各候依次 | 阴遁9-1局 |

### 排盘步骤

1. **确定节气与元** — 查当前节气属于上元/中元/下元
2. **确定阳遁/阴遁** — 冬至后阳遁，夏至后阴遁
3. **确定局数** — 根据节气与元确定1-9局
4. **布地盘** — 按局数将九宫排入地盘天干
5. **布天盘** — 以时辰天干定值符，值符随时干落宫
6. **定值符值使** — 时辰所在旬首定值符（星）和值使（门）
7. **布八门** — 值使随时干落宫，余门依序排布
8. **布九星** — 值符随时干落宫，余星依序排布
9. **布八神** — 阳遁顺排，阴遁逆排

### 信息收集

**第一轮**：求测时间（年月日时，公历或农历）
**第二轮**：求测事项（用于确定用神）
**第三轮**：相关细节（根据事项补充）

### 判断流程

1. 找到用神所在宫位
2. 看用神宫的天地盘天干关系
3. 看用神宫的门星神组合
4. 看用神宫与日干宫的生克关系
5. 看用神宫与时干宫的关系
6. 综合判断吉凶成败

## 保真机制

### 减少LLM幻觉的策略

1. **结构化排盘** — 所有要素严格按规则排布，不允许自由发挥

3. **矛盾检测** — 门克宫为凶，不可解读为吉
4. **已知案例校验** — 用历史名人事件验证排盘准确性

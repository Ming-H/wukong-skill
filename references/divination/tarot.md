## 正位与逆位

### 双重解读系统

每张牌有两种解读方向：

| 状态 | 含义 |
|------|------|
| 正位（Upright） | 牌面核心含义的直接展现 |
| 逆位（Reversed） | 能量受阻、内化、延迟或过度 |

### 逆位解读原则

1. **受阻** — 能量存在但被阻碍，需要疏通
2. **内化** — 外在表现不明显，内心体验强烈
3. **延迟** — 该能量尚未完全展现，需要等待
4. **过度** — 能量过强导致失衡
5. **拒绝** — 当事人不愿面对该能量

适用于需要明确回答的问题。

## 解读流程

### 标准流程

```
洗牌（用户默念问题）→ 抽牌（按牌阵位置）→ 逐牌解读 → 综合分析 → 给出建议
```

### 解读原则

1. **先看单牌** — 每张牌在其位置上的独立含义
2. **再看组合** — 牌与牌之间的呼应与冲突
3. **后看整体** — 牌阵的整体故事线
4. **最后建议** — 基于解读给出行动建议

### 牌面冲突处理

当牌面出现矛盾信号时：

| 情况 | 处理方式 |
|------|----------|
| 大牌vs小牌矛盾 | 以大牌为核心趋势，小牌为细节表现 |
| 正位vs逆位矛盾 | 指出张力所在，不做单方面判断 |
| 花色冲突 | 分析元素间的制约关系（如火克水） |
| 数字递变 | 关注数字序列中的趋势走向 |

### 信心标签

| 标签 | 适用场景 |
|------|----------|
| `[经典原文]` | 牌面固定含义、正逆位核心释义 |

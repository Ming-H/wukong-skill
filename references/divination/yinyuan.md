## 模式一：八字合婚

### 信息收集流程

**第一轮**：双方出生年份（获取年柱）
**第二轮**：双方出生月日（获取月柱日柱）
**第三轮**：双方出生时辰（获取时柱，可选）

### 分析维度

1. 年柱天合地合程度 `[学术共识]`
2. 日主五行互补关系 `[学术共识]`
3. 用神是否互助 `[经验推断]`
4. 夫妻宫（日支）互动关系 `[学术共识]`
5. 大运流年感情时机 `[经验推断]`

### 输出原则

- 绝不使用"不配""不宜结合"等否定性结论
- 以"需要注意的方面"替代"不合"
- 每指出一个需要注意的方面，必须附上化解建议
- 强调"命理仅为参考，经营才是关键"

## 模式四：求签系统

### 百签体系

本系统采用100签制，分为上上、上吉、中吉、中平、中下、下下六档。

### 签文结构

每支签包含：

| 要素 | 说明 |
|------|------|
| 签号 | 第X签 |
| 等级 | 上上/上吉/中吉/中平/中下/下下 |
| 签诗 | 四句七言诗 |
| 解曰 | 白话解读 |
| 仙机 | 具体指引 |
| 问姻缘 | 专门针对感情的解读 |

### 求签数字流程

1. 请用户心中默念所求之事
2. 请用户说出一个1-100之间的数字（或由用户选择三个数字取模）
3. 根据数字对应签号
4. 输出签文及解读

### 签文解读原则

- 下下签也必须给出希望和出路
- 每签的"问姻缘"部分需针对感情主题
- 不论签的好坏，都引导积极态度

## 模式五：桃花分析

### 桃花类型

| 类型 | 判定方法 | 含义 |
|------|----------|------|
| 墙内桃花 | 年支查，桃花在月支 | 正当桃花，婚姻美满 |
| 墙外桃花 | 日支查，桃花在时支 | 魅力外显，需注意分寸 |
| 正桃花 | 与日主五行相合 | 正式姻缘 |
| 偏桃花 | 与日主五行冲克 | 烂桃花/短暂情缘 |

### 桃花查询法

- 寅午戌见卯为桃花
- 巳酉丑见午为桃花
- 申子辰见酉为桃花
- 亥卯未见子为桃花

## 模式六：红线指引

### 流程

1. 收集用户基本信息
2. 结合八字分析感情时机
3. 给出方向性指引（非具体对象）
4. 提供改善姻缘的建议

## 反焦虑机制

### 核心原则

所有姻缘相关输出必须遵循以下原则：

1. **绝不判定"不配"** — 只说"需要注意的方面"
2. **绝不制造紧迫感** — 不说"再不结婚就晚了"
3. **绝不否定缘分** — 不说"你命中无姻缘"
4. **始终给出希望** — 即使不利也提供化解方案
5. **强调主观能动性** — 命理为参考，经营是关键
